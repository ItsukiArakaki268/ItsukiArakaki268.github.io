ssh-agentの使い方

 Posted on October 26, 2023  |  1 minutes  |  32 words  |  Itsuki Arakaki

GitHub に push する際、ssh-agent の使い方で躓いたのでメモがわりに。

ssh-agent の起動

$ eval `ssh-agent`

ssh-agent の停止

$ ssh-agent -k

ssh-add の使い方

秘密鍵を ssh-agent に登録。

$ ssh-add [秘密鍵のパス]

登録されている鍵の公開鍵一覧を表示。

$ ssh-add -L

登録されている鍵を削除。

$ ssh-add -d [秘密鍵のパス]
